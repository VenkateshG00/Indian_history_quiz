# Indian History Quiz

A 10-question quiz on Indian history, from the Indus Valley to independence.

**Live demo:** https://YOUR-USERNAME.github.io/indian-history-quiz/

## Features
- Three difficulty modes (easy, mixed, hard)
- Randomised questions from a 15-question bank
- Live score and progress bar
- Explanation shown after every answer
- End-of-quiz review of missed questions
- Dark mode support
- Zero dependencies — one HTML file

## Tech
Plain HTML, CSS, and JavaScript. Built by describing the app to Claude.
